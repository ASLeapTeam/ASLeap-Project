About Us

ASLeap is an interactive learning program for both the hearing and the deaf. The program teaches users American Sign Language (ASL) using the Leap Motion Controller to recognize the accuracy of each alphabet sign. The Leap Trainer recognizes hand poses and gestures as 3 dimensional coordinates that we then assign to letters of the ASL alphabet. This website serves as a teaching tool featuring games and activities that allow users to learn the ASL Alphabet in fun and interactive ways.
